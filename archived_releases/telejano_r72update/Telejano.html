Installation:

  • Unzip the files to your Quake folder with subdirectory extracting enabled. You should now have a telejano.exe, telejano.html, fiend451.jpg, and a telejano folder.

Running Telejano:

  • telejano.exe -width 640 -height 480 -conwidth 640 -bpp 16 -winmem 32
    This will load Telejano with 640x480 resolution, 16 bit color depth, and 32 megabytes of RAM allocated to memory.
  • telejano.exe -width 640 -height 480 -conwidth 640 -bpp 32 -winmem 64
    This will load Telejano with 640x480 resolution, 32 bit color depth, and 64 megabytes of RAM allocated to memory.
  • telejano.exe -width 800 -height 600 -conwidth 800 -bpp 32 -winmem 64
    This will load Telejano with 800x600 resolution, 32 bit color depth, and 64 megabytes of RAM allocated to memory.
  • telejano.exe -width 1024 -height 768 -conwidth 1024 -bpp 32 -winmem 64
    This will load Telejano with 1024x768 resolution, 32 bit color depth, and 64 megabytes of RAM allocated to memory.
  • telejano.exe -width 1600 -height 1200 -conwidth 1600 -bpp 32 -winmem 128
    This will load Telejano with 1600x1200 resolution, 32 bit color depth, and 128 megabytes of RAM allocated to memory.
  • -data <eyecandy mod folder>
    This additional command line will load a eyecandy mod (texturepack, skinpack, modelpack, soundpack, hudpack, etc..)
  • -game <modfolder>
    This additional command line will load a mod
  • +map <mapname.bsp>
    This additional command line will load a map
  • -game <modfolder> +map <mapname.bsp>
    These additional command lines will load a mod and then load a map in the mod

{

}

1
{ 
  file snow1
   delay 0.5
}

This will force a animation cicle from snow1 to snow1. You can have a maximun to 99 frames. Place your env skybox in gfx/env/ has usual.

ZONES! (beta)

Smooth transition from fog setting, rain, snow, storms, ...

Commands:

  • zone show zone settings
  • loadzone load new transitions settings from zone/ folder
  • savezone (disabled) save zone_ values in a .cfg file in the zone/ folder
  • auto2zone (disabled) copy fog values to zone values
  • zone_enable enable transition smoother
  • zone_fog* new fog settings
  • zone_start fog start
  • zone_end fog end

How work?

define new transitions status (zones) in the zone/ folder as .cfg files. load with loadzone <name>,

loadzone stormblue will load stormblue.cfg
